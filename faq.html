Tabme — FAQ

Chrome extension for organizing bookmarks and tabs.

What is Tabme, and how does it work?

Tabme is a Chrome extension designed to make managing your tabs and bookmarks effortless.
You can save tabs into folders, use drag-and-drop functionality to arrange them. Tabme also supports color-coding, archiving, search, dark mode, and keyboard accessibility to enhance your browsing experience.

How to disable/enable Chrome newtab override by Tabme?

There is two versions of tabme:
regular "Tabme" version one with override Chrome newtab,
and "Tabme — version without newtab".
Install the one of the two that suits you best.

How syncing saved bookmarks works?

Syncing saved bookmarks with chrome bookmarks and across devices is in progress.

How do I save and organize my tabs?

To save your tab, simply drag and drop it from sidebar into any folder.
If you drag-and-and tab into "New Folder" then new Folder will be created automatically.
You can assign colors to Folders for easy identification.
You can also use the "Stash" feature to save all open tabs into a single folder.

Can I open all bookmarks from a folder at once?

Yes! With the "Open All" feature, you can instantly restore all the tabs saved in a specific folder.
This is particularly useful when you want to reopen a group of stashed tabs.

How does search functionality work?

The search to quickly find any tab: currently open, saved, archived, or recently closed.
It's accessible from the keyboard.
1) You can activate search input by TAB key.
2) You can start typing immediately if the page is already focused.

What happens to my bookmarks when I archive them?

Archiving bookmarks in Tabme is a way to clean up your working space while keeping bookmarks accessible for future use. To show archived bookmarks click "Settings" → "Show archived".

Does Tabme support dark mode?

Absolutely! You can switch between light and dark modes in the "settings" menu.

Is there keyboard accessibility support?

Yes, Tabme is designed with keyboard accessibility in mind. You can navigate your bookmarks only using keyboard.
1) Open newtab in browser;
2) press TAB key to focus page;
3) and use arrow keys to navigate all bookmarks.

Is Tabme really free to use?

Yes, Tabme is completely free to use. Enjoy it without any hidden fees or subscriptions! If you like the extension I would welcome a positive review on the ChromeStore.
